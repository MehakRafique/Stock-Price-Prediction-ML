Iris Dataset Exploratory Data Analysis
Task Objective

The objective of this task is to perform basic data exploration and visualization on the Iris dataset using Python. The project aims to understand the structure of the dataset and analyze relationships between different features through graphical visualization.

Dataset Used

The dataset used in this project is the Iris dataset. It is a well-known dataset in machine learning that contains measurements of iris flowers from three different species.

The dataset includes the following features:

Sepal Length

Sepal Width

Petal Length

Petal Width

Species

The dataset is loaded directly using the Seaborn library.

Model Applied

No machine learning model is applied in this project. Instead, the focus is on Exploratory Data Analysis (EDA). The analysis is performed using Python libraries including Pandas for data handling, Matplotlib for plotting graphs, and Seaborn for advanced visualizations.

Key Results and Findings

From the visualizations and analysis, several observations can be made:

Scatter plots help show relationships between sepal length and sepal width.

The histogram of sepal length shows the distribution of values in the dataset.

The boxplot helps identify variation and potential outliers in sepal length.

The scatter plot with species differentiation shows how petal length and sepal length vary among different iris species.
